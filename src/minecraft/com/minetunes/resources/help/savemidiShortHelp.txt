SaveMIDI [MIDI name on next line]

Saves the ditty as a MIDI file in your .minecraft/midi folder. Only one SaveMIDI keyword is heeded per ditty.

The filename should be letters and numbers, with no symbols or spaces. ".mid" is automatically added to the end when saving.
